
后端提示词（Spring Boot + MyBatis Plus）

项目目标:
开发一个物流单号管理与结账系统，支持：
- 导入 Excel 表格；
- 批量抓取物流单号内容；
- 自动生成待结账数据；
- 结账筛选与导出；
- 用户管理与权限控制；
- 高性能分页与筛选展示（数据量约 8 万条）。

一、系统结构
技术栈：Spring Boot + MyBatis Plus + MySQL + Vue + Element Plus
模块划分：auth、order、settlement、admin、report、common

二、核心业务逻辑
1. 入库模块
    - Excel 导入：时间、单号、型号、SN、备注
    - 写入数据库：order_record 表
2. 对账功能
    - 批量输入物流单号
    - 抓取内容并保存待结账表
    - 支持人工录入价格
    - 二次结账警告
3. 筛选功能
    - 按日期、分类、状态筛选
    - 一键清账导出
4. 结账与导出
    - 按分类或日期范围结账
    - 导出 Excel
5. 用户与权限
    - 管理员可增删改用户
    - 普通用户仅提交/查询
    - 日志记录操作
6. 性能优化
    - 分页查询
    - 索引优化
    - 异步导出
    - Redis 缓存
7. 老板管理
    - 老板相当于用户管理员,可以生成用户账户,可以对用户增删改
8. 用户
    -用户拥有账户之后,可以提交物流单号,查看自己提交过的物流单号 
    
三、数据库设计概要
order_record 表、settlement_temp 表、sys_user 表、sys_log 表

四、接口设计
包括上传、抓取、结账、导出、用户管理、日志查询等 RESTful 接口。

五、AI 提示词模板
请基于 Spring Boot + MyBatis Plus 编写一个用于管理物流单号与结账的后端系统，包含导入、批量查询、结账、筛选、导出、用户权限、日志功能。
